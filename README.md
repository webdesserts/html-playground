# HTML Playground

## Pre-requisites

1. [node.js](http://nodejs.org/download/)
2. [gulp.js](http://gulpjs.com) `npm install -g gulp`

## Getting Started

```
git clone https://github.com/webdesserts/html-playground.git
cd html-playground
gulp
```

This will start a standard static server at `localhost:3000`.

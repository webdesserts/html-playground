# HTML Playground

## Pre-requisites

1. [node.js](http://nodejs.org/download/)
2. [gulp.js](http://gulpjs.com)(`npm install -g gulp`)

## Getting Started

```
git clone http://github.com/webdesserts/static-template.git
cd static-template
gulp
```

This will start a standard static server at `localhost:8080`.
